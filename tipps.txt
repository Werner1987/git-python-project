Aufgaben (nicht nur) lösbar mit:
--------------------------------

1) git bisect / git show:

    ./monte_carlo_pi/monte_carlo_pi.py
    ./fibonacci/fibonacci.py

    Der erste Commit ist mit "initial" getaggt. 
    D.h.: Ihr könnt folgendes machen:

    $ git bisect start HEAD initial
    $ git bisect run ...?...

2) git grep / git blame:

    ./int2hex/int2hex.py
    ./fac/fac.py

3) git branch / git checkout:

    ./euler/euler.py
    ./deduplicate/deduplicate.py

4) git log / git show:

    ./reverse/reverse.py
    ./count_even/count_even.py


Die Tests könnt ihr manuell ausführen mit:

    $ make test_<taskname>

Beispielsweise:

    $ make test_euler

(Vor dem Pushen kontrollieren!)
